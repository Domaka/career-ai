# Career AI — Proof-of-Work Intelligence Engine

An AI-powered career assistant that analyzes:
- CV
- GitHub activity
- Skills

Then generates:
- Strength analysis
- Skill gaps
- Job readiness score
- Personalized roadmap

## Tech Stack

Frontend:
- React + TypeScript
- Vite

Backend:
- Django
- Django REST Framework
- PostgreSQL

## Vision

To build a career operating system for developers based on real proof-of-work.
